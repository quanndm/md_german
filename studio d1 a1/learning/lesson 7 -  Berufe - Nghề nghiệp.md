### Modelverben: Động từ khuyết thiếu 
- Động từ khuyết thiếu sẽ chiếm vị trí thứ 2 ở trong câu, sau đó động từ chính sẽ bị đẩy xuống cuối câu (giữ nguyên thể ) 

- möchten: muốn 

|           | möchten  |
| --------- | -------- |
| Ich       | möchte   |
| du        | möchtest |
| er/sie/es | möcht    |
| wir       | möchten  |
| ihr       | möchtet  |
| Sie/sie   | möchten  |

- müssen: phải 

|           | müssen |
| --------- | ------ |
| Ich       | muss   |
| du        | musst  |
| er/sie/es | muss   |
| wir       | müssen |
| ihr       | müsst  |
| Sie/sie   | müssen |

- können: có thể 

|           | können |
| --------- | ------ |
| Ich       | kann   |
| du        | kannst |
| er/sie/es | kann   |
| wir       | können |
| ihr       | könnt  |
| Sie/sie   | können |

### Atikelwörter im Akkusativ - mạo từ trong trường hợp Akkusativ (c4)

| m   | den                | einen                | keinen              | _en              |
| f   | die                | eine                 | keine               | _e               |

| pl  | die                |                      | keine               | _e               |
